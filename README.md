# Employee Management System

## Overview

This is a backend Employee Management System built using Spring Boot. It provides REST APIs to manage employee records with CRUD operations.

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Postman

## Features

- Add Employee
- Get All Employees
- Get Employee by ID
- Update Employee
- Delete Employee

## API Endpoints

| Method | Endpoint  |
|--------|-----------|
| POST | /emp      |
| GET | /emp      |
| GET | /emp/{id} |
| PUT | /emp/{id} |
| DELETE | /emp/{id} |

## Database

- MySQL
- Database Name: `employee_management`

- IntelliJ IDEA
